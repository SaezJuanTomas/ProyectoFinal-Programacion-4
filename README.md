# INTEGRADOR FOOD STORE

Backend de proyecto final para gestion de productos, categorias e ingredientes.

## Requisitos

- Python 3.10+
- Entorno virtual (.venv)

## Instalacion

1. Crear y activar entorno virtual.
2. Instalar dependencias del proyecto.

## Ejecucion

- Ejecutar la aplicacion principal desde `main.py`.

## Estructura principal

- app/: codigo de la aplicacion
- scripts/demo/: scripts de demostracion
- test_architecture.py: prueba de arquitectura

## Notas

- El proyecto incluye `docker-compose.yml` para entorno con contenedores.
